70 /**
71  * Wait for an interrupt
72  *
73  * This function waits for an rising edge interrupt and then verifies that
74  * the interrupt pin is high. If it is not high it will wait again for
75  * another interrupt. This extra loop makes sure that this function does
76  * not return true if the interrupt pin is low (could be caused by a
77  * previous interrupt that occurred before or during sensor initialization).
78  *
79  * This function asserts that the pin previously has been initialized as
80  * GPIO_DIR_INPUT_INTERRUPT. It also asserts that all gpio operations
81  * are successfully.
82  *
83  * @param[in] pin The pin.
84  * @param[in] timeout_ms Maximum time in milliseconds to wait for an interrupt
85  *
86  * @return true if an interrupt was received within timeout
87  */
